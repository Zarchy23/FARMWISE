## System Overview

FarmWise is a comprehensive agricultural management system designed for Zimbabwean farmers, cooperatives, and agricultural stakeholders. The system provides end-to-end farm management capabilities from crop planning to market sales.

### Key User Types
- **Smallholder Farmer**: Manages individual farms
- **Large Scale Farmer**: Manages multiple large farms
- **Cooperative Administrator**: Manages farmer cooperatives
- **Agronomist/Extension Officer**: Provides technical expertise
- **Equipment Owner**: Rents out farm equipment
- **Insurance Agent**: Manages crop/livestock insurance
- **Market Trader**: Buys/sells agricultural products
- **Veterinarian**: Provides animal health services
- **Soil Lab Technician**: Analyzes soil samples
- **Supermarket**: Agricultural retail/wholesale
- **System Administrator**: Manages the platform

## Technical Stack

### Backend
- **Framework**: Django 5.2.15
- **Database**: PostgreSQL
- **Task Queue**: Celery + Redis
- **API**: Django REST Framework
- **Authentication**: Django Allauth + JWT

### Frontend
- **Styling**: Tailwind CSS
- **Icons**: Remix Icons
- **Maps**: Leaflet
- **Charts**: Chart.js
- **Reactivity**: Alpine.js
- **Real-time**: Django Channels

### AI/ML
- **ML Library**: scikit-learn 1.5.2
- **Deep Learning**: TensorFlow 2.16.1
- **Data Processing**: pandas, numpy
- **AI APIs**: OpenAI, Groq, Google Generative AI

### Deployment
- **Server**: Gunicorn
- **Static Files**: WhiteNoise
- **Storage**: AWS S3 (django-storages)
- **Email**: SendGrid
- **SMS**: Twilio
- **Payments**: Stripe
